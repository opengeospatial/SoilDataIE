Appears In: Soil - Description
A data type that describes the particle size distribution, and physical and chemical properties of the various soil size fractions (such as 'fine soil material', 'whole soil', 'coarse fragments'), either within a soil horizon or an undifferentiated soil. These properties were taken from the GlobalSoilMap consortium’s requirements.
